This is the PEFIT 65*65 grid version.

Compiling PEFIT:

1. The source code of PEFIT is placed in ~/pefit_65/ folder.
2. Typing "make".
3. The executable file is in the ~/pefit_65/bin/ folder.

Running PEFIT:

1. Go to folder ~/pefit_65/bin/.
2. The green table and input data are all in the ~/pefit_65/bin folder.
   PEFIT reads the diagnostics from the file rtscDigdata.dat.
3. Typing "./test".
4. The outputs on the screen are the currents in the PF coil and the plasma current parameters.
